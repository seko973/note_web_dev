Command Line Techniques and Directory Navigation

ls : permet de liste le contenu d'un répertoire
cd nomRepertoire : Permet de changer de localisation
cd ~ : retour au root directory
cd .. : remonte d'un cran
cd /Users/Seko/Documents/Learn/Languages : Astuces pour arriver directement sur le caractère "D" de "Documents" par exemple
NB: ALT + Mouse click on the "D"

CTRL + A : aller au début
CTRL + E : aller à la fin
CTRL + U : efface la commande affichée
clear : clear la console

Creating, Opening, and Removing Files through the Command Line

pwd : print writing directory. Affiche le répertoire courant
mkdir nomDuRepertoireCree : to make directory
touch fileName.txt : to create un new file
start fileName.txt : to open the file with the default program
start Code fileName.txt : to open the file with Visual Studio Code
rm fileName.txt : to remove a file
rm * : efface tous les fichiers
rm -r nomRepertoire/ : efface le répertoire
