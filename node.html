Utilisation de node

Node s'utilise avec l'hyper terminal
Ex: Node index.js exécute le fichier index.js depuis l'hyper terminal

The Node REPL (Read Evaluation Print Loops)

si je tape simplement node dans l'hyper terminal
alors on accède au REPL (prononcé RIPLE). Tout se passe alors comme si on était en mode console sur Chrome Dev Tools
On a un prompt >
Et on peut taper des commandes JS à la volée

Pour sortir du REPL :

  • ".exit" + ENTREE
  • CTRL + C 2 fois

How to Use the Native Node Modules

La première chose à faire avant d'utiliser un module est de le réclamer
const fs = require('node:fs/promises'); from Node file system docs

fs.copyFileSync("file1.txt", "file2.txt"); //copie un fichier

The NPM Package Manager and Installing External Node Modules

NPM = Node Package Manager

depuis l'hyper terminal taper : npm init
NB: à faire une fois pour le projet en cours

Ensuite il faut rechercher son package sur https://www.npmjs.com/
Puis il faut installer le package à partir de l'hyper terminal
Ex: npm install superheroes

Puis il faut écrire le code de réclamation et de traitement dans le fichier "index.js"
var superheroes = require('superheroes');
var mySuperheroName = superheroes.random();
console.log(mySuperheroName);
