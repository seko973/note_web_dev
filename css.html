CSS

NB : pour trouver de belles couleurs utiliser le site colorhunt.co

Exemple de CSS dans le code HTML

NB: Utilisation de l'attribut style="code css avec : au lieu de = en HTML"

<body style="background-color: #FEFCF3;">

Internal CSS pour un fichier HTML

Se positionne dans le head du code HTML

<head>
 <style>
  body {
   background-color: #FEFCF3;
  }
 </style>
</head>

NB: il existe un Browser Default CSS appliqué sur les éléments HTML
Il suffit de le taper dans google pour obtenir du détail à partir du premier lien
Cela aide à comprendre les effets inatendus suite à l'application du CSS

Le plugin pesticide pour chrome permet d'afficher les zones des éléments HTML d'une page

External CSS pour plusieurs fichiers

Il faut créer un fichier ".css" comme par exemple styles.css
NB: On le place généralement dans un sous-répertoire /css

On place le code CSS dans le fichier
Exemple :

body {
 background-color: #FEFCF3;
}

hr {
 border-color: #DFD3C3;
 border-style: none;
 border-top-style: dotted;
 border-width: 5px;
 width: 3%;
}

Puis on place dans le head des fichiers concernés un élément HTML de type "link" :

<head>
 <link rel="stylesheet" href="css/styles.css">
</head>
