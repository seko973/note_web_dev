Exemple de CSS dans le code HTML

NB: Utilisation de l'attribut style="code css avec : au lieu de = en HTML"

<body style="background-color: #FEFCF3;">

Internal CSS

Se positionne dans le head du code HTML

<head>
 <style>
  body {
   background-color: #FEFCF3;
  }
 </style>
</head>
