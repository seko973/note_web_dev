External CSS pour plusieurs fichiers

Il faut créer un fichier ".css" comme par exemple styles.css

On place le code CSS dans le fichier
Exemple :

}

Puis on place dans le head des fichiers concernés un élément HTML de type "link" :

<head>
 <link rel="stylesheet" href="css/styles.css">
</head>

Grammar of the CSS language

selector { CSS rules }
selector {
 rule1;
 rule2;
}

NB: une "best practice" est d'organiser les rules par ordre alphabétique
pour retrouver une règle plus facilement dans un lot important

selector { property : value ; }
selector = who ?
property = what ?
value = how ?

NB: le who et value est le plus facile à déterminer.
En revanche, pour le what mieux vaut se référer ici

Class attribute of HTML

l'attribut "class" permet de cibler les effets du CSS sur un ensemble d'éléments HTML dans une page
<img class="bacon" src="https://em-content.zobj.net/thumbs/120/apple/325/bacon_1f953.png" alt="bacon-img">

Selector = .NomDeLaClasse
.bacon { background-color: green; }

On peut indiquer plusieurs classes sur un même élément HTML
<img class="broccoli circular" src="https://em-content.zobj.net/thumbs/120/apple/325/broccoli_1f966.png" alt="broccoli-img">

Code CSS correspondant

.broccoli {
 background-color: red;
}

.circular {
 border-radius: 100%;
}

id attribute of HTML

l'attribut "id" permet de cibler les effets du CSS sur un unique élément HTML dans une page
<h1 id="heading">I Love Bacon</h1>

Selector = #NomID
#heading { color: green; }

NB: C'est le style le plus précis qui l'emporte en cas de conflit

  1. id
  2. class
  3. élément
  4. default css

HTML Divs

Les div sont des éléments HTML qui n'ont aucun sens si il ne sont pas associés au CSS
Ils permettent de structurer/compartimenter les éléments d'une page

The Box Model of WebSite styling

Chaque élément est traité comme une box par le CSS
On peut styliser la box en modifiant :

  • sa largeur
  • sa marge (margin)
  • son remplissage (padding)
La modification de la marge ne modifie pas la taille du contenu. Si on l'augmente ça déplace le contenu sans effet sur sa taille
La modification du padding en revanche oui. Si on l'augmente ça déplace et réduit le contenu

La méthode de design consiste à :
  1. Créer des divs dans la page html
  2. Les colorer avec du CSS
  3. Jouer sur les variables de la box au niveau de l'outil de dev de Chrome
  4. reporter les valeurs retenues dans le code CSS du projet
NB: Chrome dev tools joue un rôle crucial dans cette phase de conception

CSS Display property

La propriété dsiplay peut prendre 4 valejurs :

  • block
  • inline
  • inline-block
  • none
par défaut, il existe des éléménts affichés en ligne (inline) :
  • parapgraphe (<p>)
  • Headers (<h1> to <h6>)
  • Division (<div>)
  • Listes et éléménts de liste (<ol>, <ul> et <li>)
  • Formulaires (<form>)
NB: les éléménts inline ont une hauteur et une largeur qui s'adaptent automatiquement au contenu
Impossible de modifier/forcer les valeurs de width ou height

NB: les éléménts inline-block sont disposés en ligne avec un contrôle possible sur la largeur et la hauteur

NB: les éléments none disparaissent. C'est comme si ils n'existaient pas dans le code HTML

Et, il existe des éléments affichés par défaut en bloc (block) :
  • Spans (<span>)
  • Images (<img>)
  • Liens (<a>)
NB: l'élément HTML span n'a aucun intérêt sans le CSS associé (Comme le div)

On peut changer l'affichage par défaut d'un élément en jouant sur la proprété "diplay"
Ex: diplay: inline; sur un paragraphe <p>

visibility: hidden;
Permet de rendre un élément invisible mais il continue à occuper la place qu'il devrait dans la page

CSS Positionning

La propriété "position" peut prendre 4 valeurs :

  • static
  • relative
  • absolute
  • fixed
static: est le positionnement par défaut de tous les éléments HTML

relative : déplace l'élément relativement par rapport à sa position statique
img {
 position: relative;
 left: 30px;
}
L'image se déplace de 30px par rapport à son bord gauche. Il s'agit donc d'un déplacement vers la droite
NB: un déplacement relatif n'affecte pas le positionnement des autres éléments.
tout se passe comme si l'élément occupait sa place statique

absolute : déplace l'élément relativement par rapport à son parent
div {
 position: relative;
}
img {
 position: absolute;
 left: 30px;
}
L'image se déplace de 30px par rapport au bord droit de son conteneur. Le conteneur doit avoir une position relative
Si la position relative n'est pas précisée, alors le parent relatif devient le body.
NB: une position absolute sort l'élément du flow HTML. une position absolute est une position relative au parent

fixed : fixe l'élément par rapport à l'écran indépendamment du scroll.

Centering elment with CSS

text-align: center;
suffit généralement
Mais text-align ne donne pas le résultat attendu sur un élément de type block réduit (de 30% par exemple)
Dans ce cas, il faut utiliser :
margin: 0 auto 0 auto;
sur l'élément de type block réduit

Font styling

Il existe 2 types de polices de base :

  • Serif
  • Sans-Serif
NB: Serif = empattement = lettre avec des terminaisons en forme de pied

font-family: verdana, sans-serif;
Si on veut se donner le maximum de chance d'afficher la police souhaitée il faut utiliser les web safe font
En effet, il se peut que certains navigateur ou machine n'affiche pas la font attendue
Ce site permet de copier le code CSS avec des secours
Ex: font-family: "Helvetica Neue", Helvetica, Arial, sans-serif;

Si on veut forcer l'utilisation d'une font, on peut l'intégrer. Le site google fonts permet de prendre les font qui nous interesse
Il suffit ensuite de coller ceci dans le head du code HTML :
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Merriweather&family=Montserrat&family=Sacramento&display=swap" rel="stylesheet">

Et ceci dans le code CSS :
font-family: 'Merriweather', serif;
font-family: 'Sacramento', cursive;
font-family: 'Montserrat', sans-serif;

Il vaut mieux exprimer la font-size en valeur dynamique
Ex: font-size: 562.5%; ou bien font-size: 5.625em; plutôt que font-size: 90px;
Cela permet de ne pas figer la taille de la police quand on active certaines fonctionnalités des navigateurs
NB: une font-size de 100% équivaut à 16px et équivaut à 1em
body {
 font-size: 2em;
}
h1 {
 font-size: 5em;
}
NB: Attention! Dans ce cas de figure la taille du texte est démultiplié par 10 = 2 x 5 x 16px
Idem pour les pourcentage
Cela n'arrivera pas quand la taille est exprimée en pixel. La configuration la plus précise écrase le reste
Pour éviter les surprises, il faut utiliser font-size: 5rem; avec r comme root

CSS Floating and clear

float: left;
Détermine qu'un élément est dans une position flottante sur la gauche
Les autres éléments vont s'organiser sur la droite et envelopper l'élément flottant

clear: left;
Annule une position décalé en raison d'un élément flottant. Clear s'appliquent aux éléments qu'on ne veut pas voir se décaler.
clear: left; signifie annule le margin de gauche positionné à cause de l'élément flottant à gauche

NB: le float est à utiliser avec prudence. Seulement quand on a pas le choix et qu'on veut enrouler du texte autour d'un autre élément

CSS Z-index

z-index: 0; est la valeur CSS par défaut des éléments HTML
On peut jouer sur cette propriété CSS pour faire passer un élément devant l'autre. Ex: z-index: 10;
NB: Cela ne fonctionnera qu'entre élément possédant une position différente de position: static; soit la position par défaut des éléments HTML
