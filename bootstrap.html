Bootstrap Grid layout system

Grid system
Le principe :
Chaque ligne comporte 12 items
un div class="row"
qui encadre des div class="col"
div class="col-6" signifie qu'on occupe la moitié de l'écran

si on veut que le résultat soit responsive, on utilise lg, md, sm dans le nom des classes
<div class="row">
 <div class="col-lg-3 col-md-4 col-sm-6">
...
Cela signifie que dans une ligne chaque colone occupe jusqu'à 3 cases en mode desktop
4 cases en mode tablette
6 cases en mode smartphone

  • lg = large
  • md = medium
  • sm = small

Bootstrap Containers

Les conteneurs sont des éléments responsives
ils centrent leur contenu et permettent d'ajouter du padding
div class="container"
le container possède un max-width qui varie selon le viewport

div class="container-fluid"
Le container-fluid occupe toute la largeur de l'écran
