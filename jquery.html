jQuery

jQuery est une bibliotèque JS. C'est la plus populaire.
Elle raccourcit considérablement le code
$(); = document.querySelector(); = jQuery();
$("h1").css("color","red");

Incorporate jQuery to Websites

Utiliser le CDN le plus populaire (celui de google). On le trouve sur le site w3schools.com
Le CDN se place de préférence juste avant le body pour des raisons d'optimisation de la performance

Le code jQuery est généralement placé dans le fichier index.js du projet
Attention à ne jamais placer le CDN de jQuery après ce fichier. Autrement on aura une erreur

$(document).ready(function() {
//jQuery code
});
Ce code signifie que l'on attend le chargement complet du fichier JS de jQuery avant d'exécuter le code
NB: ce code est important si on place le CDN dans le head de la page

Minification to Reduce File Size

Il s'agit de retirer tous les espaces et retour à la ligne pour gagner des octets dans le fichier et reduire le temps de chargement

Selecting Elements with jQ

$("h1"); contrairement à querySelector() le code jQuery va sélectionner tous les éléments correspondant
NB: il n'y a pas d'array à considérer

Manipulating Styles

$("h1").css("color","red"); applique la valeur à la propriété sélectionnée

var color = $("h1").css("color"); to get the property value

Selon le même principe que le JS, il faut privilégier les modifications de classe (définie dans le CSS) pour modifier le style à partir du jQuery
$("h1").addClass("huge"); ou bien $("h1").addClass("huge margin-50"); pour plusieurs classes
$("h1").removeClass("huge"); ou bien $("h1").removeClass("huge margin-50");

NB: $("h1").hasClass("huge"); renvoie true ou false

Manipulating Text

$("h1").text("My new text");
$("h1").html("<em>My new text</em>");

Manipulating Attributes

$("img").attr("src"); get attribute
$("a").attr("href","https://kelou.fr"); set attribute

Adding Event Listener

$("h1").click(function(){
 //do something
});

On appelle la methode correspondant au listener et on passe en paramètre une callback function
La liste complète des methodes se trouve ici : https://www.w3schools.com/jquery/jquery_ref_events.asp

Une autre façon de faire et qui se rapproche de la notation JS :
$("h1").on("click",function(){
 //do something
});
https://www.w3schools.com/jsref/dom_obj_event.asp Documentation MDN sur les type d'event

Add, Remove and Hide Elements with JS

Ajout d'un bouton avant le "h1"
$("h1").before("<button></button>");

Ajout d'un bouton après le "h1"
$("h1").after("<button></button>");

Ajout d'un bouton dans le "h1" avant le texte du "h1"
$("h1").prepend("<button></button>");
NB: le bouton est entre "<h1> et </h1>"

Ajout d'un bouton dans le "h1" après le texte du "h1"
$("h1").append("<button></button>");
NB: le bouton est entre "<h1> et </h1>"

Supprimer un élément : $("button").remove(); supprime tous les boutons de la page

Cacher un élément : $("h1").hide(); l'élément disparait comme avec un position: none; en CSS
$("h1").show(); pour le faire réapparaitre
$("h1").toggle(); pour basculer automatiquement de l'état hide vers show et inversement

Websites Animation with jQ

Animation pour cacher les éléments :

$("h1").fadeOut(); Fait disparaitre l'élément avec un effet de fondu
$("h1").fadeIn(); Fait apparaitre l'élément avec un effet de fondu
$("h1").fadeToggle(); pour basculer automatiquement de l'état fadeOut vers fadeIn et inversement

$("h1").slideUp(); Fait disparaitre l'élément avec un effet de glissement
$("h1").slideDown(); Fait apparaitre l'élément avec un effet de glissement
$("h1").slideToggle(); pour basculer automatiquement de l'état slideUp vers slideDown et inversement
NB: slideToggle() est un effet utilisé pour les dropdown menu par exemple

Animation personnalisée :

$("h1").animate({opacity: 0.5}); change l'opacité en douceur
$("h1").animate({margin: 100px}); ajoute de la marge en douceur
NB: cette methode ne fonctionne qu'avec des propriétés qui attendent des valeurs numériques
Ca ne fonctionne pas avec color ou backgroud-color par exemple

Enchainer les animations :

$("h1").slideUp().slideDown().animate({opacity: 0.5});
https://www.w3schools.com/jquery/jquery_ref_effects.asp pour une liste complète des effets possibles
