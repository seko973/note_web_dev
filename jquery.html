jQuery

jQuery est une bibliotèque JS. C'est la plus populaire.
Elle raccourcit considérablement le code
$(); = document.querySelector(); = jQuery();
$("h1").css("color","red");

Incorporate jQuery to Websites

Utiliser le CDN le plus populaire (celui de google). On le trouve sur le site w3schools.com
Le CDN se place de préférence juste avant le body pour des raisons d'optimisation de la performance

Le code jQuery est généralement placé dans le fichier index.js du projet
Attention à ne jamais placer le CDN de jQuery après ce fichier. Autrement on aura une erreur

$(document).ready(function() {
//jQuery code
});
Ce code signifie que l'on attend le chargement complet du fichier JS de jQuery avant d'exécuter le code
NB: ce code est important si on place le CDN dans le head de la page

Minification to Reduce File Size

Il s'agit de retirer tous les espaces et retour à la ligne pour gagner des octets dans le fichier et reduire le temps de chargement

Selecting Elements with jQ

$("h1"); contrairement à querySelector() le code jQuery va sélectionner tous les éléments correspondant
NB: il n'y a pas d'array à considérer

Manipulating Styles

$("h1").css("color","red"); applique la valeur à la propriété sélectionnée

var color = $("h1").css("color"); to get the property value

Selon le même principe que le JS, il faut privilégier les modifications de classe défini dans le CSS pour modifier le style à partir du jQuery
$("h1").addClass("huge"); ou bien $("h1").addClass("huge margin-50"); pour plusieurs classes
$("h1").removeClass("huge"); ou bien $("h1").removeClass("huge margin-50");

NB: $("h1").hasClass("huge"); renvoie true ou false

Manipulating Text

$("h1").text("My new text");
$("h1").html("<em>My new text</em>");

Manipulating Attributes

$("img").attr("src"); get attribute
$("a").attr("href","https://kelou.fr"); set attribute
